# 🚀 Zero-Node Math Engine

Un ecosistema interactivo de experimentación técnica construido bajo la filosofía **Zero-Node**: sin dependencias, sin instaladores pesados y optimizado para hardware de bajos recursos.

### 🛠️ Tecnologías Utilizadas
- **Lenguajes:** HTML5, CSS3, Vanilla JavaScript (ES6+).
- **Arquitectura:** Zero-Node (HTML-First).
- **Conceptos Implementados:** - **IA:** Perceptrón simple (Red Neuronal de 1 capa).
  - **Criptografía:** Algoritmo de hashing determinista manual.
  - **Big Data:** Visualización de algoritmos de ordenamiento (Bubble Sort).
  - **Ciberseguridad:** Sistema de auditoría y persistencia en LocalStorage.

### 📈 Características
- **Ultra-Ligero:** Todo el proyecto pesa < 50KB.
- **Privacidad:** Procesamiento 100% en el lado del cliente (Local-First).
- **Educativo:** Visualización paso a paso de lógica matemática compleja.
